## Правила ведения (Karpathy principle 5)

Подробно в [`AGENTS.md`](./AGENTS.md). Кратко:
1. **Один файл на сущность.** Глубокий, не обзорный.
2. **Каждый факт** — со ссылкой на источник (чат, дата).
3. **Без догадок.** Если не знаю — `status: unknown`.
4. **Не перезаписывать** без версионирования.
5. **В ответе Сергею** — только дифы и ссылки.
6. **«Запомни» / «добавь в вики»** — немедленное обновление.
7. **`lint wiki`** — ревизия.
