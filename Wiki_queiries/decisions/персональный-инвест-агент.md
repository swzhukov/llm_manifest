# Решение: построить персональный инвестиционный агент (2026-06)

> **Источник:** [`../projects/персональный-инвест-агент.md`](../projects/персональный-инвест-агент.md).
> **Связанные чаты:** `05_DS` (2026-06-08), `07_DS` (2026-06-06).

---

## Решение

Сергей решил построить **персонального инвестиционного агента** на базе n8n, который:
1. Учитывает сделки через Telegram (текст/голос).
2. Формирует ежемесячный план инвестиций.
3. Считает P&L и даёт рекомендации.
4. Собирает еженедельный дайджест сигналов из Telegram + YouTube.

---

## Контекст

- Портфель Сергея ~480k₽ через Альфа-Инвестиции.
- **ИИС не пополнялся 2 года** — потеря вычетов.
- Целевая структура 40/35/25 (стабильный/рост/рискованный).
- Сергей хочет дисциплину (15k₽/мес) + автоматизацию.

---

## Технический стек

| Слой | Инструмент |
|------|-----------|
| Оркестратор | n8n (self-hosted на Beget) |
| Интерфейс | Telegram Bot |
| БД | Google Sheets (5 листов) |
| LLM | YandexGPT (агент + надсмотрщик) |
| Голос | Whisper API |
| Котировки | MOEX ISS API |
| YouTube | RSS + YouTube Data API v3 |
| RPA (опц.) | Pix RPA |

---

## MVP (4 недели)

| Неделя | Что |
|--------|-----|
| 1 | Google Sheets — 5 листов по схеме |
| 2 | WF4A — Telegram-дайджест (быстрая ценность) |
| 3 | WF1 — Учёт сделок (Telegram → YandexGPT → Sheets) |
| 4 | WF2 + WF3 — Дисциплина и Отчёт |

---

## Ключевая фишка

**Надсмотрщик (supervisor)** — классический паттерн Зуевича. Агент генерирует, надсмотрщик проверяет и фильтрует.

---

## Что зависит от Сергея

- **Написать код** workflows в n8n (или скопировать шаблон n8n #6317).
- **Создать Google Sheets** с 5 листами.
- **Настроить Telegram-бота** и каналы для дайджеста.
- **Возможно, пополнить ИИС** на 400k₽ за 2026 (для вычета 52k₽).

---

## Главный риск

- **Не реализовано** (на 2026-06-16). ТЗ есть, кода нет.
- **Pix RPA** для Альфа-Инвестиций — хрупкий, может сломаться.
- **YandexGPT** может быть слабее GPT-4 на финансовых формулировках.
