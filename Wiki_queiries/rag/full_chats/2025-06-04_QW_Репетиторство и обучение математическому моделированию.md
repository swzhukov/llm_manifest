        *   *Производная (dydx):* Скорость изменения. **Пример:** В модели роста бактерий `dN/dt = k * N` (`N` - количество, `t` - время, `k` - скорость роста). Производная `dN/dt` показывает, *как быстро* меняется `N` прямо сейчас.
        *   *Интеграл (∫):* Накопление эффекта. **Пример:** Зная скорость роста `dN/dt`, интеграл дает общее количество бактерий `N(t)` через время `t`.
        *   *Векторы/Матрицы:* **Пример (Вектор):** Состояние системы "Киоск" = [Продажи_сегодня, Запас_мороженого]. **Пример (Матрица):** Если киосков много, матрица связей покажет, как спрос в одном влияет на другой.

*   **Теория & Примеры:**
    *   **Состояние системы:** Вектор значений *прямо сейчас*, определяющих будущее (e.g., `[Положение, Скорость]` мячика).
    *   **Метод Эйлера (Численное решение ДУ):** Идея: Зная скорость изменения (`dy/dt`) сейчас, прикинем новое состояние через маленькое время `dt`: `y_new = y_old + dy/dt * dt`.
    *   **Модель Распада (Аналог кофе):** `dN/dt = -λ * N` (λ - постоянная распада). **Решение Эйлером:**

        import matplotlib.pyplot as plt

        N0 = 1000  # Начальное количество
        lam = 0.1  # Постоянная распада (1/мин)
        T_total = 30  # Общее время (мин)
        dt = 0.5    # Шаг времени (мин)
        # Массивы для результатов
        time_points = np.arange(0, T_total + dt, dt)
        N_euler = np.zeros_like(time_points)
        N_euler[0] = N0
        # Цикл Эйлера
        for i in range(1, len(time_points)):
            dNdt = -lam * N_euler[i-1]  # Вычисляем производную в предыдущей точке
            N_euler[i] = N_euler[i-1] + dNdt * dt  # Новое значение
