🏢 СПРАВОЧНАЯ ИНФОРМАЦИЯ: СТРУКТУРА ДКП5

ЧЕТЫРЕ ОСНОВНЫХ ЦФО:

| ЦФО | Руководитель | Статус | Доход | Норма маржи | ВЗР-модель |
|-----|-------------|---------|-------|-------------|------------|
| УКП 5.1 | Карина | ЦФО P&L | 100% ERP-проектов | 30-35% | Платит ВЗР за программистов |
| СЛЭЙ | Оля Третьякова | ЦФО P&L | SLA + консультации | 35-40% | Платит ВЗР за специалистов |
| ОР | Саша Балан | ЦФО P&L | ТОР-разработки | 25-30% | Платит ВЗР за программистов |
| Центр РЦ | Галина | Ресурс | ВЗР от других ЦФО | N/A | Должен покрывать 100% потребностей |

КЛЮЧЕВЫЕ ВЗАИМОЗАВИСИМОСТИ:
- ВЗР-дефицит = Центр разработки не может поставить программистов в полном объеме → УКП 5.1 и ОР теряют маржу
- Встречи в СЛЭЙ = Low-margin работа, съедает маржу = нужна автоматизация
- Трудозатраты = Коррелируют с выручкой + указывают на выгорание и неэффективность

📊 ВХОДНЫЕ ДАННЫЕ И СТРУКТУРА

А. ФИНРЕЗ ФАЙЛЫ (Excel) - ожидаемая структура:
```
Строки (показатели по месяцам):
├─ 1. Поступление денежных средств, руб
├─ 2. Сумма реализации (выручка по актам), руб
│  ├─ 2.1 Реализация услуг (основной доход)
│  ├─ 2.2 Реализация ПО
│  └─ 2.3 Реализация оборудования
├─ 3. Себестоимость реализации, руб
├─ 4. Взаиморасчеты с другими отделами (ВЗР)
├─ 5. Прибыль / Убыток
├─ 6. Маржа (%) = (Реализация - COGS - ВЗР) / Реализация × 100%
└─ 7-10. Расходы (зарплаты, администрирование, прочее)

Колонки (месяцы):
[Месяц1: С НДС Факт | Без НДС | План]
[Месяц2: С НДС Факт | Без НДС | План]
...
[Текущий месяц: С НДС Факт | Без НДС | План]

⭐ ФОКУС: Работаем с "Без НДС" (чистые цифры)
```

Б. ТРУДОЗАТРАТЫ ФАЙЛ (Excel) - ожидаемая структура:
```
Колонки:
├─ День (дата)
├─ ФИО сотрудника
├─ Тип работы (Проектная / Администрирование / Совещания / Обучение / Внутренние)
├─ Описание работы
├─ Проект (название или ID)
└─ Всего часов (числовое значение)

Ожидаемое количество месячных данных: 21 рабочий день × 50+ сотрудников
```

### 1.4 КРОСС-АНАЛИЗ: ВЫРУЧКА VS ВЗР vs МАРЖА

Матрица проблем (красные флаги):
```
Если ВЫРУЧКА ↓ И ВЗР ↑ → ДВОЙНОЙ УДАР (маржа падает вдвое)
Если МАРЖА ↓ И COGS ↑ → Управленческая проблема (высокие затраты)
Если ВЗР > ВЫРУЧКА → УБЫТОК (ЦФО платит больше чем зарабатывает)
Если ВЫРУЧКА растет, но МАРЖА ↓ → Неправильно растет (на низко-маржинальных продуктах)
```
