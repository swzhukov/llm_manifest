## AI

Для получения количества записей в регистре накоплений "СебестоимостьТоваров" можно использовать несколько вариантов запросов:

## Вариант 1: Простое получение количества записей
```1c
ВЫБРАТЬ
    КОЛИЧЕСТВО(*) КАК КоличествоЗаписей
ИЗ
    РегистрНакопления.СебестоимостьТоваров
```
*Примечание: для периодического регистра будет выбрана таблица оборотов.*

## Вариант 2: С явным указанием таблицы оборотов
```1c
ВЫБРАТЬ
    КОЛИЧЕСТВО(*) КАК КоличествоЗаписей
ИЗ
    РегистрНакопления.СебестоимостьТоваров.Обороты
```

## Вариант 3: Для остаточного регистра (если это регистр остатков)
```1c
ВЫБРАТЬ
    КОЛИЧЕСТВО(*) КАК КоличествоЗаписей
ИЗ
    РегистрНакопления.СебестоимостьТоваров.Остатки
```

## Вариант 4: С фильтрацией по периоду и измерениям
```1c
ВЫБРАТЬ
    КОЛИЧЕСТВО(*) КАК КоличествоЗаписей
ИЗ
    РегистрНакопления.СебестоимостьТоваров КАК Регистр
ГДЕ
    Регистр.Период МЕЖДУ &НачалоПериода И &КонецПериода
    И Регистр.Номенклатура = &Номенклатура
```

## Вариант 5: Получение количества записей по разрезам
```1c
ВЫБРАТЬ
    Регистр.Номенклатура,
    КОЛИЧЕСТВО(*) КАК КоличествоЗаписей
ИЗ
    РегистрНакопления.СебестоимостьТоваров КАК Регистр
ГДЕ
    Регистр.Период МЕЖДУ &НачалоПериода И &КонецПериода
СГРУППИРОВАТЬ ПО
    Регистр.Номенклатура
```

**Рекомендация:** Если регистр периодический, используйте Вариант 1 или 2. Для остаточного регистра - Вариант 3. Если нужны дополнительные условия фильтрации - Вариант 4 или 5.
