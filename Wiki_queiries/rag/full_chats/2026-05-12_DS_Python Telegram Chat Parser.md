## user

Режим генерации: строго детерминированный. Температура 0.2, top_p 0.85, top_k 20. Приоритет — корректность синтаксиса, отсутствие пояснительного текста вне кода.

Роль: Ты — старший Python-разработчик, специализирующийся на обработке структурированных данных и автоматизации в Google Colab.

Задача: Написать полностью готовый к запуску Python-скрипт, который обрабатывает стандартный экспорт Telegram Desktop (чат или канал) и формирует единый структурированный файл.

Входные данные:
- Директория, содержащая один или несколько файлов result*.json (result.json, result_1.json и т.д.)
- Стандартный формат JSON, экспортируемый через Telegram Desktop → Экспорт истории чата

Требования к скрипту:
1. Среда исполнения: Google Colab. Использовать только стандартные библиотеки Python и предустановленный pandas.
2. Загрузка данных: автоматический поиск всех файлов, соответствующих маске result*.json, в указанной директории. Последовательное чтение и объединение массивов сообщений.
3. Парсинг полей:
   - text: корректная обработка случаев, когда поле является строкой или списком объектов формата Telegram
   - date / date_unixtime: приведение к единому формату, сортировка по возрастанию времени
   - from / from_id: извлечение имени отправителя и идентификатора, обработка отсутствующих значений
   - media_type, file/photo/document/video: извлечение типа вложения и имени файла
   - reply_to_message_id, forward_from, pinned, edited: флаги и ссылки
4. Структура результата: единый pandas DataFrame с явно заданными типами данных, без пропусков в ключевых столбцах, отсортированный хронологически.
5. Экспорт: сохранение в CSV с кодировкой utf-8-sig (для корректного открытия в Excel). Предусмотреть переменную конфигурации для выбора формата (csv/json/parquet). Реализовать скачивание результата через google.colab.files.download.
6. Надёжность: обработка ошибок чтения JSON, пропуск записей без поля message_id, логирование количества обработанных файлов и строк, конфигурационные переменные в начале скрипта.
7. Формат ответа: предоставить только готовый код. Код должен выполняться в одном блоке без интерактивных запросов. Комментарии — лаконичные, на русском языке.

Ограничения:
- Не использовать сторонние пакеты, требующие установки через pip.
- Не выводить пояснительный текст вне блока кода.
- Учесть, что в Colab файловая система временная; скрипт должен работать с абсолютными или относительными путями, заданными в переменных.

Сгенерируй код в соответствии с указанными требованиями.
