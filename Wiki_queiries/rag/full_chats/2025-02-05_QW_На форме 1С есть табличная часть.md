# На форме 1С есть табличная часть

## USER

На форме 1С есть табличная часть с колонками "Товар, количество, цена, сумма", напиши программный код на языке 1С, который бы рассчитывал сумму в строке при изменении количества или цены

---

## AI

Вот пример кода для обработки изменения количества или цены и автоматического расчета суммы в строке табличной части:

```bsl
&НаКлиенте
Процедура КоличествоПриИзменении(Элемент)
    ОбновитьСуммуСтроки(Элемент.Данные);
КонецПроцедуры

&НаКлиенте
Процедура ЦенаПриИзменении(Элемент)
    ОбновитьСуммуСтроки(Элемент.Данные);
КонецПроцедуры

&НаКлиенте
Процедура ОбновитьСуммуСтроки(СтрокаТабличнойЧасти)
    Если СтрокаТабличнойЧасти = Неопределено Тогда
        Возврат;
    КонецЕсли;
    
    СтрокаТабличнойЧасти.Сумма = СтрокаТабличнойЧасти.Количество * СтрокаТабличнойЧасти.Цена;
КонецПроцедуры
```

Пояснения:

1. Обработчики событий `КоличествоПриИзменении` и `ЦенаПриИзменении` вызываются при изменении соответствующих полей в таблице

2. Оба обработчика вызывают общую процедуру `ОбновитьСуммуСтроки`, передавая текущую строку таблицы через `Элемент.Данные`

3. В процедуре `ОбновитьСуммуСтроки`:
   - Проверяем, что строка существует
   - Рассчитываем сумму как произведение количества на цену
   - Записываем результат в колонку "Сумма"

Для работы этого кода необходимо:

1. В табличной части должны быть реквизиты:
   - Товар (СправочникСсылка.Номенклатура)
   - Количество (Число)
   - Цена (Число)
   - Сумма (Число)

2. В форме должны быть назначены обработчики событий:
   - Для поля "Количество" - событие `ПриИзменении` → `КоличествоПриИзменении`
   - Для поля "Цена" - событие `ПриИзменении` → `ЦенаПриИзменении`

3. Поле "Сумма" должно быть:
   - Либо вычисляемым в форме
   - Либо иметь возможность записи (если это поле табличной части)

Дополнительно можно добавить:
- Округление суммы до 2 знаков после запятой
- Обработку нулевых значений

Пример с округлением:
```bsl
СтрокаТабличнойЧасти.Сумма = Окр(СтрокаТабличнойЧасти.Количество * СтрокаТабличнойЧасти.Цена, 2);
```
