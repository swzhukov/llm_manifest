| :--- | :--- | :--- |
| **Что это** | Готовый сервис от n8n | Вы сами устанавливаете и запускаете n8n на своем оборудовании или у облачного провайдера [citation:2][citation:8] |

### 🚀 Способ 1: Облачный бесплатный план (Самый простой)

Это самый быстрый способ попробовать n8n без каких-либо настроек.
1.  Перейдите на [официальный сайт n8n](https://n8n.io/).
2.  Зарегистрируйтесь, выбрав бесплатный тариф.
3.  Сразу после регистрации вы получите доступ к онлайн-редактору и сможете создавать свои первые автоматизации.
4.  В бесплатной версии вам даже будут доступны **бесплатные кредиты OpenAI** для экспериментов с AI-функциями, что отлично подходит для обучения [citation:9].

### 💻 Способ 2: Самостоятельное размещение (Self-hosting) (Максимум возможностей)

Этот вариант открывает все функции n8n бесплатно. Вам нужно будет установить его на каком-либо устройстве. Вот несколько популярных и действительно **бесплатных** способов это сделать:

*   **На свой компьютер (для тестов):** Самый простой способ попробовать self-hosting — запустить n8n локально. Если у вас установлен [Docker](https://www.docker.com/products/docker-desktop/), просто выполните эту команду в терминале [citation:1][citation:4][citation:9]:
    ```bash
    docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n
    ```
    После этого n8n будет доступен в браузере по адресу `http://localhost:5678`. Этот способ идеален для разработки и тестирования, но работает только пока включен ваш компьютер.

*   **На бесплатный виртуальный сервер (VPS) у облачных провайдеров:** Это позволит вам иметь постоянно работающий n8n, доступный из любой точки, и при этом не платить ни копейки. Главное — не превышать лимиты бесплатного использования.
    *   **На Google Cloud:** Можно воспользоваться их **Always Free** предложением и создать виртуальную машину (например, тип `e2-micro`). Затем установить Docker и запустить n8n [citation:1][citation:4].
    *   **На Oracle Cloud:** У них очень щедрый **Always Free** уровень с мощными ARM-серверами (до 4 ядер OCPU и 24 ГБ RAM!). Этого с запасом хватит для любых задач n8n [citation:3].

*   **На свой домашний сервер (например, Raspberry Pi):** Если у вас есть устройство, которое работает 24/7 (например, старый компьютер или Raspberry Pi), вы можете установить n8n на него. Чтобы безопасно открыть к нему доступ из интернета, можно использовать **Cloudflare Tunnel** — это бесплатно и безопасно [citation:7].

### ⚠️ Важное предупреждение о "бесплатности" self-hosting

Хотя лицензия на ПО бесплатна, важно понимать, что поддержание production-сервера (который обрабатывает важные для бизнеса данные) может потребовать ресурсов и знаний. В статьях, сравнивающих n8n с управляемыми платформами, реалистично оценивают **скрытые расходы** [citation:5][citation:10]:

*   **Время и знания:** Вам нужно будет самостоятельно обновлять n8n, следить за безопасностью сервера, настраивать бэкапы. Если ваш сервер "упадет" посреди ночи, чинить его придется вам.
*   **Ресурсы сервера:** Для серьезных нагрузок может потребоваться более мощный (и уже платный) сервер. Оценки затрат на полноценный production-сервер (с учетом администрирования) могут достигать **$200–$500 в месяц** [citation:5][citation:10].

**Итог:** Для личного использования или небольших проектов self-hosting на любом из описанных бесплатных способов — идеальное и абсолютно бесплатное решение. Для бизнес-критичных задач стоит либо закладывать бюджет на администрирование, либо рассмотреть готовый платный облачный план n8n, чтобы сэкономить свое время.
