# Практический курс «Агентный инженер за 2 недели»

## Модуль 1. Фундамент (дни 1–3)
- **День 1.** Установка AGENTS.md.
- **День 2.** Тренировка «точечных изменений».
- **День 3.** Ритуал «думай прежде чем кодить».

## Модуль 2. LLM Wiki (дни 4–7)
- **День 4.** Инициализация вики.
- **День 5.** Ежедневное наполнение «запомни это».
- **День 6.** Использование вики для ответов.
- **День 7.** `lint wiki` – аудит.

## Модуль 3. Loop Engineering (дни 8–11)
- **День 8.** Создать Maker–Checker.
- **День 9.** Добавить условие остановки.
- **День 10.** Состояние на диске.
- **День 11.** Внедрение в реальную работу.

## Модуль 4. Интеграция (дни 12–14)
- **День 12.** Утренний ритуал.
- **День 13.** Вечерний ритуал.
- **День 14.** Итоговое испытание.

## Чек-лист для самопроверки
- [ ] В каждом проекте есть `AGENTS.md`.
- [ ] Я не пишу промпт без фразы «перескажи задачу».
- [ ] У меня есть LLM Wiki с папками `raw/`, `wiki/`.
- [ ] Я выполняю `lint wiki` не реже 1 раза в неделю.
- [ ] Я построил хотя бы один цикл Maker–Checker.
