# Сравнение веб-клиентов LLM

## Таблица суперсил
| Сервис | Суперсила | Идеальный сценарий | Ограничения |
|--------|-----------|---------------------|--------------|
| **DeepSeek** | Глубокое мышление (R1) | Сложные задачи, код | Нет мультимодальности |
| **Kimi** | 200万字 контекста | Анализ больших документов | Медленнее на коротких задачах |
| **Minimax** | 4 млн токенов | Мега-проекты | Избыточен для простых задач |
| **Qwen** | Мультимодальность | Работа с картинками, схемами | Логика уступает R1 |
| **Perplexity** | Актуальный поиск | Фактчекинг, обзоры | Требует интернета |

## Стратегия «оркестратора»
1. **Perplexity** – найти источники.
2. **Kimi / Minimax** – сжать в таблицу.
3. **DeepSeek (R1)** – решить логическую задачу.
4. **Qwen** – визуализировать результат.

## Ограничения веб-клиентов
- **Нет API** → нельзя автоматизировать evals.
- **Слабый контекст проекта** → создавайте `CONTEXT.md`.
- **Многозадачность проваливается** → разбивайте на шаги.
