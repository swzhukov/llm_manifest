# Правила для AI-агентов (по Карпатому)

1. **Think before coding** – перескажи задачу и перечисли допущения.
2. **Simplicity first** – минимальное решение, без избыточных абстракций.
3. **Surgical changes** – изменяй только то, что относится к задаче.
4. **Goal-driven** – сформулируй измеримый критерий успеха.

## Дополнительные правила для вики
- Никогда не перезаписывай файл целиком без сохранения версии.
- При добавлении термина в `glossary.md` убедись, что на него есть ссылка из `topics/`.
- **Frontmatter:** всегда сохраняй YAML-блок в начале файла при создании или обновлении.
- Команда `lint wiki` – приоритетна.
